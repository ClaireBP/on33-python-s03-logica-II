# Atividade de Verificação de Número e Texto


## Descrição

Este é um projeto de atividade de casa da terceira semana do curso de Python: análise de dados solicitado pela professora Jenifer Plácido da parte de Lógica II 

O projeto usa de inputs do usuário para realizar duas tarefas: reconhecer um número como par ou ímpar e fazer uma contagem de caracteres de uma palavra ou frase, utilizando-se de 'if', 'else', 'try' e 'except'.

## Implementação

A partir do uso de uma função desenvolvida será verificado se o número inteiro fornecido pelo usuário é par ou ímpar. Caso não seja um número inteiro, o sistema retornará a entrada "Por favor, digite um número inteiro!" através do uso do except. Logo em seguida, será feita a contagem de caracteres de um palavra ou frase também solicitada. Caso não seja identificada como palavra ou frase, o sistema retornará a entrada "Erro, digite uma palavra ou frase!" também através do uso do except. Assim, quando o valor da quantidade de caracteres for identificado, será mostrado se esse valor é um número par ou ímpar.



## Uso do Projeto

O usuário dá entrada em dois inputs:

- o primeiro, um número inteiro qualquer;
  - O sistema mostrará se esse número é par ou ímpar.
- o segundo, uma palavra ou frase.
  - O sistema contará quantos caracteres possui. E mostrará também se esse numero é par ou ímpar.
